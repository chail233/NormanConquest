## 规则
- 初始时，双方具有一定的`HP`，当`HP`归零后，该方失败，另一方获胜。
- 双方拥有各自的`牌堆`与`弃牌堆`。双方从各自的`牌堆`抽牌，初始时各拥有5张牌，每回合开始时将手牌抽到与`HP`相等。
- 每方每回合只能进行一次`通常进攻`，即打出`部队牌`。除此之外还有一种`进攻`：`特殊进攻`，`特殊进攻`没有次数限制。
- 当一方对另一方打出`部队牌`，即为一次`进攻`。通常来讲，被进攻方对于此次`进攻`打出任意一张`部队牌`便可`抵御`此次`进攻`，防止扣除`HP`。如果不`抵御`此次`进攻`，便会扣除1`HP`。
- `部队牌`有着不同的`类型`，不同的`类型`间存在`克制`关系。当被进攻方打出`克制`进攻方部队的`部队牌`时，不仅可以`抵御进攻`，还可以形成`追击`。`追击`视作一次`特殊进攻`。当然，对方也可以`抵御`或`追击`此次`追击`。
- 用于`抵御进攻`的部队若被进攻部队`克制`，则打出后无法`抵御进攻`。
- `指令牌`是一种打出后便可以立即触发`效果`的牌。
- `建筑牌`打出后将保留在自己的`建筑区`从而长久的提供`效果`。但`建筑区`最多只能保留当前`HP-2`数量的牌。当`建筑区`满后打出`建筑牌`，需要从`建筑区`选择一张牌弃掉。
- `非衍生牌`被`打出`或`弃牌`后将进入`弃牌堆`。当`牌堆`的牌被抽尽时，扣除1`HP`然后将`弃牌堆`所有牌洗入`牌堆`。
- 不同的`领袖`有着不同的`特殊能力`。
## 卡牌
卡组一共40张牌

- 部队（24）
    1. 轻步兵*8
    2. 重步兵*3
    3. 弓弩兵*5
    4. 轻骑兵*4
    5. 重骑兵*2
    6. 枪兵*3
- 指令（9）
    1. 征召：抽2张`部队牌`
    2. 土地开发*2：抽2张牌
    3. 征税*2：加1`HP`
    4. 分封：减1`HP`，抽3张牌
    5. 倾巢出动：打出手牌中所有的`部队牌`，每打出一张视作一次`特殊进攻`，并且不会被`追击`
    6. 假装撤退：随机弃掉一张`部队牌`，下回合开始时，抽2张牌
    7. 绝罚：随机弃掉对方一张手牌
- 建筑（7）
    1. 庄园：每回合开始抽1张牌
    2. 城墙：当被进攻成功时，使此次进攻失败，然后将此牌弃掉
    3. 城堡：每回合通常进攻次数加一
    4. 马厩：己方的骑兵若成功进攻，将其返回手牌
    5. 兵营：进攻成功造成的伤害加一
    6. 教堂：进攻成功时，加1HP
    7. 市场：己方弃牌时，抽1张牌

### 克制关系
- 轻步兵->()
- 重步兵->(轻步兵，枪兵)
- 弓弩兵->(轻步兵，枪兵)
- 轻骑兵->(弓弩兵)
- 重骑兵->(轻步兵，重步兵，弓弩兵)
- 枪兵->(轻骑兵，重骑兵)
